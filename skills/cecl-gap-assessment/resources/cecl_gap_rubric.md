# CECL Gap Assessment Rubric

## Minimum checks

- determine whether execution-based review is blocked
- compare documented segment structure to output structure
- compare scenario narrative to numeric scenario tables
- compare overlay documentation to provided reserve support
- issue concrete evidence requests

## Red flags

- no reserve engine or reproducibility evidence
- no lineage linking outputs to scenarios and model version
- scenario narrative not supported by numeric scenarios
- overlay documentation materially understates reserve effect
