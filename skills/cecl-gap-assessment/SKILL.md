# CECL Gap Assessment

Use this skill when the CECL package is documentation-heavy or runtime-blocked.

## Core Review Behaviors

- assess whether execution-based review is possible
- compare methodology, model overview, and scenario materials
- reconcile documented segments to provided outputs
- assess whether overlay support is quantitatively sufficient
- produce evidence requests instead of implying unsupported execution

## Minimum Outputs

- blocked coverage statement
- documentation cross-check
- evidence request list
- stakeholder-facing gap assessment
