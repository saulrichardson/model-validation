# CECL Final Report

Use this skill to synthesize a CECL review into a stakeholder-facing deliverable.

## Rules

- do not produce notebook-style logs as the hero output
- stakeholder-facing outputs should be concise, bank-facing, and opinionated
- report only what the evidence supports
- keep supporting detail in appendices or Markdown support artifacts

## Deliverable Shapes

- model-driven workflow: CECL review memo
- documentation-led workflow: CECL gap assessment
