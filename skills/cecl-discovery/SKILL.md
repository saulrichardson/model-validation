# CECL Discovery

Use this skill when reviewing a CECL package.

## Objective

Identify whether the package supports:

- a model-driven CECL review
- a documentation-led CECL gap assessment

## Discovery Checklist

- inventory model code or reserve-process implementation
- inventory scenario tables and prior outputs
- identify documentation covering methodology, model overview, overlays, and governance
- identify data lineage and reproducibility evidence
- identify missing evidence that blocks execution-based review

## Output Expectations

- concise discovery summary
- explicit coverage posture
- list of key artifacts and key gaps
