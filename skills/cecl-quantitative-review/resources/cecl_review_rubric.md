# CECL Quantitative Review Rubric

## Minimum checks

- baseline reproduction against supplied outputs
- baseline/adverse/severe reserve reruns
- segment-level reserve comparison
- sensitivity to forecast horizon
- sensitivity to reversion speed
- sensitivity to overlay magnitude
- reasonableness of severe versus adverse outcomes

## Red flags

- documented horizon or reversion does not match implementation
- severe scenario is not directionally worse where expected
- overlays are more material than described in governance materials
- baseline cannot be reproduced within tolerance
