# CECL Quantitative Review

Use this skill for runnable CECL packages.

## Core Review Behaviors

- reproduce the supplied baseline reserve output
- rerun baseline, adverse, and severe scenarios
- compare reserves overall and by segment
- run sensitivity tests on horizon, reversion, macro severity, and overlays
- identify directionally odd segment behavior
- connect results back to documentation claims

## Minimum Outputs

- baseline reproduction summary
- scenario comparison table
- sensitivity summary
- reserve driver bridge
- findings grounded in files and outputs
