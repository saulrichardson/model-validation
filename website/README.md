# Website

Self-contained public-facing website for the model validation platform.

Everything for the public site lives in this folder so the rest of the repo can evolve independently.

## Commands

```bash
pnpm install
pnpm dev
pnpm build
pnpm start
```

## Notes

- Built as a standalone Next.js app for Vercel deployment.
- Current scope is the public homepage only.
- Product claims are grounded in the existing `agent-gateway` workbench artifacts and demo flows.
