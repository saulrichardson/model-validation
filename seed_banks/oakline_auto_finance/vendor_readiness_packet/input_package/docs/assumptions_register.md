# Model Assumptions Register (Partial)

1. Model assumes current indirect auto applicant mix is representative of scoring development data.
2. Model is assumed to meet Oakline's baseline fair lending and bias controls, though no supporting evidence was provided.
3. Features not included in the partial dictionary (dealer quality band, loan term stretch flag) are assumed to be formatted per vendor's correspondence.
4. Vendor API response times and uptime are assumed to meet service-level requirements; evidence for this was requested but not provided.

*Note: No explicit vendor-supplied documentation of statistical or business assumptions. Several key operational and performance assumptions remain unverified.*
