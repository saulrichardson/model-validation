# Material Change Request

**Change Type:** Material – Full Revalidation

**Initiator:** Model Risk Management Team, Atlas Community Bank 

**Change Summary:**
- Recalibration (lowering) of underwriting intercept and threshold to reflect portfolio drift during 2024
- Incorporation of "low_cash_buffer" feature to address observed liquidity stress in new origination cohorts
- Realignment of reason code mapping: one new code (LOW_CASH_BUFFER), minor changes to THIN_CREDIT_FILE and SHORT_EMPLOYMENT_TENURE weights
- Coverage and calibration evaluated at updated standards (>0.85)

**Rationale:**
- To improve discrimination and address recent trends in liquidity-driven distress
- Additional monitoring of reason code frequency, especially under simulated downturn conditions

**Committee Approval:** 2025-01-08
